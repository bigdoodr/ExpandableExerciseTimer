# Expandable Exercise Timer
A flexible xOS workout timer app that lets you create custom exercise routines with timed or rep-based exercises, configurable rest periods, and audio alerts.
Features

- Custom Exercise Routines - Add, remove, and rearrange exercises to build your perfect workout
- Flexible Exercise Types - Set each exercise as time-based or rep-based
- Configurable Rest Periods - Specify how long to rest after each exercise
- Timer Controls - Pause, resume, or cancel your routine at any time
- Audio Alerts - Get notified with sound when each timer expires
- Screen Wake - Keep your device screen active during workouts
- Import/Export - Save and share your workout routines
- Completion Notifications - Receive a notification when your routine finishes

## Use Cases

- HIIT (High-Intensity Interval Training)
- Circuit training
- Strength training routines
- Tabata workouts
- CrossFit WODs
- Custom fitness routines

## Requirements

- iOS/iPadOS 18.6 or later
- macOS 15.6 or later
- visionOS 26.2 or later
